## 📌 발표 개요

**지정 주제**: 출판단지 지역 활성화를 위한 AI 문화 및 콘텐츠 서비스

## 🎯 문제 정의

### 현재 문제점

1. **문화 공간 활용도 저조**
   - 출판단지, 갤러리, 문화센터 등 방문객 부족

2. **프로그램-시간 불일치**
   - 문화 행사가 생활인구 패턴과 맞지 않아 참여도 저조

3. **개인화 부족**
   - 획일적인 문화 안내로 개인 맞춤 추천 어려움

4. **데이터 활용 부족**
   - 방대한 생활인구·소비 데이터가 있으나 활용되지 않음

## 💡 서비스 개요

### PAJU Culture Lab

**"머신러닝이 데이터를 분석하고, AI가 맞춤형 문화 여정을 만듭니다"**

- **머신러닝 기반 데이터 분석**: 12개 예측 모델로 프로그램 추천
- **생성형 AI 큐레이션**: 업스테이지 Solar Pro2로 개인화 추천
- **실시간 대시보드**: 관리자용 분석 인터페이스

---

## 🚀 핵심 기능

### 1. 당장 실행할 일
- 머신러닝 예측 기반 즉시 실행 가능한 액션 아이템 (10-12개)
- 날짜별 특성 반영 (주말/평일, 계절, 공휴일)

### 2. LLM 분석 리포트
- 출판단지 활성화 종합 분석 리포트
- 서술형 분석, 강점/개선점, 실행 계획

### 3. AI 분석 어시스턴트(채팅)
- 큐레이션 및 프로그램 기획 관련 대화형 질문 답변
- 머신러닝 예측 결과 기반 인사이트 제공

### 4. 출판단지 활성화 분석
- 문화 공간 활성화 점수 (접근성, 관심도, 잠재력, 활용도)
- 출판단지 활성화 지수 (날짜별 동적 계산)
- 성연령별 타겟팅, 주말/평일 패턴 분석

---

## 🤖 AI 기술 활용

### 머신러닝 기술

**12개 큐레이션 머신러닝 모델**
- 4개 프로그램 타입 × 3개 지표
  - 추천 점수 (업종별 매출 기반)
  - 시간대별 적합도 (주말/평일, 오전/오후/저녁)
  - 타겟 고객층 매칭 (성연령별 방문 패턴)

**모델 성능**
- Random Forest / Gradient Boosting
- k-fold 교차 검증
- **R² 0.99 이상** (99% 이상 설명력)

### 생성형 AI 기술

**업스테이지 Solar Pro2**
- 머신러닝 예측 결과 기반 종합 분석
- 개인화 큐레이션 및 전략 생성
- 실행 가능한 액션 아이템 생성

### 하이브리드 AI 파이프라인

```
1단계: 머신러닝 예측 (정량적 데이터 분석)
   ↓
2단계: 데이터 요약 (예측 결과 구조화)
   ↓
3단계: 생성형 AI 분석 (LLM 종합 분석)
   ↓
4단계: 개인화 추천 (액션 아이템 생성)
```

## ✨ 창의성 및 차별성

### 창의성

1. **머신러닝 + 생성형 AI 하이브리드**
   - 정량적 예측 + 정성적 분석 결합

2. **큐레이션 중심 머신러닝 모델**
   - 문화 프로그램 큐레이션에 특화된 12개 모델

3. **날짜별 동적 특성 반영**
   - 주말/평일, 계절, 공휴일 등 시간적 맥락 고려

1. **데이터 기반 과학적 추천** (R² 0.99 이상)

3. **실행 가능한 액션 아이템** (10-12개 구체적 제안)
4. **출판단지 활성화 특화** (특화 지표 계산)
